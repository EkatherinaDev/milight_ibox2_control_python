# Сценарий для Салюта: включение и выключение MiLight

Этот документ содержит готовые сценарии для голосового ассистента **Салют**:
включение и выключение MiLight.

Важно: Салют не может напрямую запускать локальный файл `MiLightToggle.exe` на Windows.
Чтобы сценарии работали, MiLight должен быть добавлен в умный дом Салюта как устройство
`MiLight`, `Свет MiLight` или `Свет`. Для этого нужен мост: Home Assistant, backend,
MQTT-to-Cloud, Cloud-to-Cloud или другая интеграция, которая принимает команду от Салюта
и отправляет UDP-команду в MiLight iBox2.

## Сценарий 1: включить MiLight

```text
Название:
Включить MiLight

Когда:
Голосовая команда: включи milight
Голосовая команда: включи милайт
Голосовая команда: включи свет
Голосовая команда: включи свет в комнате

Тогда:
Устройство: MiLight / Свет MiLight
Действие: Включить
```

Команды голосом:

```text
Салют, включи милайт
Салют, включи milight
Салют, включи свет
Салют, включи свет в комнате
```

## Сценарий 2: выключить MiLight

```text
Название:
Выключить MiLight

Когда:
Голосовая команда: выключи milight
Голосовая команда: выключи милайт
Голосовая команда: выключи свет
Голосовая команда: выключи свет в комнате

Тогда:
Устройство: MiLight / Свет MiLight
Действие: Выключить
```

Команды голосом:

```text
Салют, выключи милайт
Салют, выключи milight
Салют, выключи свет
Салют, выключи свет в комнате
```

## Рекомендуемая настройка устройства

В умном доме Салюта лучше назвать устройство одним из коротких вариантов:

```text
MiLight
Свет MiLight
Свет
Свет в комнате
```

Если в доме несколько источников света, используйте более точное имя, например
`Свет MiLight` или `Свет в комнате`, чтобы Салют не путал устройства.

## Что должно делать устройство

Для сценария `Включить MiLight` интеграция должна отправлять команду:

```text
light_on / on
```

Для сценария `Выключить MiLight` интеграция должна отправлять команду:

```text
light_off / off
```

В текущем приложении MiLight GUI свет управляется через MiLight iBox2 по UDP-порту `5987`.
По умолчанию используется зона `0` (`all`, все зоны).
